# Asistente de Nutrición GPT

Este es un asistente básico de nutrición diseñado para proporcionar recetas saludables y contar calorías de diversos alimentos.

## Características
- **Recetario**: Una colección de recetas saludables con instrucciones paso a paso.
- **Contador de Calorías**: Información nutricional básica de alimentos comunes.
- **Interfaz de Chat**: Un asistente interactivo para consultas rápidas.

## Estructura del Proyecto
- `data/`: Contiene los archivos JSON con la información de recetas y calorías.
- `src/`: Código fuente del asistente.
- `main.py`: Punto de entrada de la aplicación.
